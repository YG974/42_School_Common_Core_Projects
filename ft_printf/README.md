# ft_printf
## Purpose
The main purpose of this project is to reproduce the printf function from the Standard C library.

- Supported conversions: `%c`, `%s`, `%p`, `%i`, `%d`, `%u`, `%x`, `%X`
- Supported flags: `#`, `+`, `(space)`
- Supported options: `-`, `0`, `.`, `*`, `width`
